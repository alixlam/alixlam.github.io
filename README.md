# Alix Lamouroux personal website

This is a static personal academic / machine-learning portfolio website designed for GitHub Pages.

## Files

- `index.html` — main website
- `assets/css/style.css` — visual styling
- `assets/files/Alix_Lamouroux_CV.pdf` — downloadable CV

## How to publish on GitHub Pages

1. Create a GitHub repository named `alixlamouroux.github.io`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.
7. Your website should appear at `https://alixlamouroux.github.io`.

## Before publishing

Replace the placeholder links in the Contact section:

```html
<a href="#">GitHub</a>
<a href="#">LinkedIn</a>
<a href="#">Google Scholar</a>
<a href="#">ORCID</a>
```

with the real URLs.

You can also add a personal photo by replacing the initials avatar in `index.html` with an image.
